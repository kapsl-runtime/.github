# 🚀 Kapsl

**Next-generation runtime for efficient AI model deployment**

Kapsl is a high-performance runtime designed to make deploying, running, and scaling AI models faster, lighter, and more efficient.

---

## ✨ What we’re building

- ⚡ Multi-models inference runtime  
- 🧠 Optimized for modern AI workloads (LLMs, vision, multimodal)  
- 📦 Lightweight deployment across cloud & edge  
- 🔧 Developer-friendly SDKs and tooling  

---

## 🧩 Repositories

- **kapsl-engine** – Core runtime (Rust)  
- **kapsl-sdk** – Developer SDK  
- **kapsl-benchmarks** – Performance evaluations  
- **kapsl-extensions** – Plugins and integrations  

---

## 🎯 Vision

We believe AI deployment should be:

- **Fast** — minimal latency, maximum throughput  
- **Simple** — no complex infrastructure required  
- **Portable** — run anywhere (cloud, local, edge)
